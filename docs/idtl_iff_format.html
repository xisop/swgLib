Indexed Triangle list record used by Star Wars Galaxies
by Ken Sewell

The indexed triangle list (IDTL) record stores vertices and triangle indices.

Type Size(bytes) Name Description

float 12 (xyz) Vertex n xyz
FourCC 4 Record Tag "INDX"
uint32 4 Record size Length of record (Excluding size of Record Tag and Record size fields.

int32 4 Vertex index 2 Every 3 consecutive indices describe one triangle.

int32 4 Vertex index n Total number if vertex indices is equal to (INDX record size / 4)
